## 🏗️ 系统概览

### 当前版本：v1.0
**已完成功能：**
- ✅ 用户登录与注册
- ✅ 工作日志看板
- ✅ 撰写工作日志
- ✅ 个人月报
- ✅ 团队月报

### 技术栈
- **前端：** React + TypeScript + Vite + Shadcn UI + Tailwind
- **后端：** Hono (Supabase Edge Functions)
- **数据存储：** Supabase KV Store
- **认证：** Supabase Auth

### 项目结构
```
TraceFlow/
├── src/                          # 前端源码
│   ├── app/
│   │   ├── components/
│   │   │   ├── ui/              # Shadcn 组件
│   │   │   ├── WorkLogBoard.tsx # 看板页
│   │   │   ├── CreateWorkLog.tsx # 创建日志
│   │   │   ├── MonthlyReport.tsx # 个人月报
│   │   │   ├── TeamMonthlyReport.tsx # 团队月报
│   │   │   └── ...
│   │   └── App.tsx
│   └── main.tsx
├── supabase/
│   └── functions/server/        # 后端服务
│       ├── index.tsx            # API 路由
│       └── kv_store.tsx         # 存储封装
└── Technical Documentation/     # 本文档目录
    └── ... (所有文档)
```
